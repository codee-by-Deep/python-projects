### 7. 🎬 IMDB Top 1000 — Data Analysis
Exploratory data analysis on IMDB's top 1000 movies with visualizations.
- **Concepts:** Pandas, Matplotlib, data cleaning, groupby, filtering
- **Dataset:** [IMDB Movies Dataset — Kaggle](https://www.kaggle.com/datasets/harshitshankhdhar/imdb-dataset-of-top-1000-movies-and-tv-shows)
- **File:** `imdb_analysis.ipynb`

**Key Findings:**
- The Shawshank Redemption is the highest rated movie (9.3)
- Drama is the most common genre (85 movies)
- Christopher Nolan has the highest average rating among directors with 5+ movies (8.46)
- Movies from the 2010s earn nearly double the gross of 1970s films
